The Project

About SmishGuard AI

An open, intelligent system built to protect users from SMS-based digital threats.

SmishGuard AI is a web-based threat detection platform that uses machine learning to classify SMS messages as legitimate, spam, or smishing in real time. Built to raise digital safety awareness, it is designed to be fast, transparent, and accessible to everyone.

Key Features

AI-Powered Scanning

SVM + TF-IDF pipeline delivers accurate, production-grade SMS classification without a GPU.

Real-time Results

Predictions returned in under 50 ms via a lightweight REST API — no page reloads required.

Accessible Interface

Clean, keyboard-navigable UI designed for non-technical users and researchers alike.

Scalable Architecture

Decoupled frontend and model API built for painless deployment, extension, and scaling.
