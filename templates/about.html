Project Overview

About SmishGuard

An NLP-based smishing detection system built with scikit-learn, TF-IDF, and Flask — designed to protect users from SMS phishing attacks in real time.

01 What is Smishing?

Smishing (SMS + Phishing) is a form of social engineering where attackers send deceptive text messages to trick recipients into revealing sensitive information, clicking malicious links, or making fraudulent payments. Attacks are crafted with urgency cues designed to bypass critical thinking.

Common attack tactics
  • Fake bank account suspension alerts
  • Prize or lottery win notifications
  • Package delivery re-scheduling scams
  • Government agency impersonation
  • Malicious shortened URL links
How SmishGuard detects them
  • TF-IDF vectorisation of message text
  • LinearSVC pattern classification
  • Trained on 5,971 labelled SMS messages
  • Sub-50 ms inference per message
  • 98.4% accuracy on held-out test set

04 Limitations & Caveats
  • The model is not 100% accurate. Always verify suspicious messages manually before taking any action.
  • Trained primarily on English-language SMS messages — performance on other languages may be significantly reduced.
  • Novel smishing techniques not present in the training data may evade detection (zero-day attacks).
  • Dataset class imbalance (~81% legitimate messages) may bias the model toward classifying borderline cases as safe.
  • This tool is for educational and demonstration purposes only — not a substitute for professional security tooling.
