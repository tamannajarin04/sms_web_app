05 Preprocessing Pipeline
1
Normalise label casing
Apply str.lower().str.strip() so variants like "Smishing" and "smishing" collapse to a single canonical form.
2
Map to binary target
Encode labels: ham → 0, all spam and smishing variants → 1. Reduces 5 classes to a clean binary classification task.
3
Drop unknown rows
Remove any rows where the binary target is NaN — these correspond to unrecognised label variants not in the mapping.
4
Drop structural columns
Discard URL, EMAIL, and PHONE columns. Text content alone achieves over 98% accuracy, making these redundant.
5
Stratified train-test split
80/20 split with stratify=y to preserve the natural class imbalance (~81% ham) in both train and test sets.
6
TF-IDF vectorisation
Fit vectoriser on training set only, then transform both splits. Prevents data leakage from the test distribution.
