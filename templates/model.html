TF-IDF Vectorizer Config
from sklearn.feature_extraction.text import TfidfVectorizer vectorizer = TfidfVectorizer( # Remove English stop words (the, is, at...) stop_words='english', # Consider 1-gram and 2-gram token combinations ngram_range=(1, 2), # Keep only top 10,000 features by frequency max_features=10000, # Ignore terms that appear in > 95% of docs (too common) max_df=0.95, # Ignore terms that appear in < 2 documents (too rare) min_df=2, )
Why LinearSVC?
Speed
Trains and infers in milliseconds — ideal for real-time SMS classification.
High Dimensionality
Excels with sparse TF-IDF matrices where feature count exceeds sample count.
Generalization
Strong regularization prevents overfitting on limited smishing samples.
Best in Class
Outperformed Naive Bayes, Logistic Regression, and Random Forest on this task.
